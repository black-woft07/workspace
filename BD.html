1.¿Que es una base de datos?

Una base de datos es un sistema digital que permite almacenar, organizar, gestionar y recuperar grandes volúmenes de información de manera estructurada y eficiente. Actúa como un contenedor electrónico donde los datos (palabras, números, imágenes) se estructuran para facilitar su acceso, actualización y análisis, a menudo utilizando un Sistema de Gestión de Bases de Datos (DBMS).

2.¿Para que sirve una base de Datos?

Una base de datos sirve para almacenar, organizar, gestionar y recuperar grandes volúmenes de información de forma sistemática, segura y rápida. Permite a empresas y aplicaciones estructurar datos en tablas (filas/columnas) para consultarlos, modificarlos o eliminarlos fácilmente, mejorando la eficiencia operativa y facilitando la toma de decisiones basada en datos.

3.¿Que es un CRUD? y definiciones de cada letra

CRUD es un acrónimo fundamental en el desarrollo de software que representa las cuatro operaciones básicas de almacenamiento persistente: Create (Crear), Read (Leer), Update (Actualizar) y Delete (Borrar). Estas acciones permiten gestionar datos en bases de datos (SQL/NoSQL) y aplicaciones, siendo esenciales para funciones como registros, perfiles y tiendas online.

Definiciones de cada letra:

Create (Crear): Genera nuevos registros o entradas de datos en la base de datos (ej: registrar un nuevo usuario).

Read (Leer/Consultar): Permite visualizar, consultar o buscar datos existentes sin modificarlos (ej: ver el perfil de un usuario).

Update (Actualizar): Modifica o edita información de registros que ya existen (ej: cambiar la contraseña de un usuario).

Delete (Borrar/Eliminar): Suprime o elimina datos almacenados de forma permanente o lógica (ej: dar de baja una cuenta).

4.¿Que es un campo?

Un campo es la unidad básica de información, representando una columna en una tabla que almacena un tipo de dato específico (como nombre, edad o fecha) para todos los registros. Define la estructura y el tipo de dato, permitiendo organizar, validar y gestionar información de manera eficiente.

5.¿Que es un registro?

Un registro es una unidad única de información estructurada, compuesta por un conjunto de campos (datos específicos como nombre, edad, id) que describen una entidad (persona, objeto, transacción) y se almacena como una fila en una tabla. También conocidos como filas o tuplas, permiten organizar y relacionar datos de manera eficiente.

6.¿Que es un dato?

Un dato es la unidad mínima de información, representada por símbolos, números o letras, que describe un hecho, entidad o característica específica sin tener contexto por sí mismo. Organizados en tablas, estos registros (como nombres, fechas, precios o números) permiten a los sistemas informáticos almacenar, procesar y estructurar información para convertirla en conocimiento útil.

7.¿que es un diagrama entidad relacion?

Un diagrama entidad-relación (DER o ERD) es una herramienta visual que representa la estructura de una base de datos, mostrando entidades (objetos/conceptos), sus atributos (características) y cómo se interrelacionan entre sí. Actúa como un plano conceptual que facilita el diseño, optimización y entendimiento de los datos en sistemas de información.
