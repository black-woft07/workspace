  ¿Que es la firma de una función en JavaScript?

La firma de una función en JavaScript es la forma en que se define: incluye el nombre de la función y los parámetros que recibe. Describe qué nombre tiene la función y cuántos argumentos espera. Ejemplo:
function sumar(a, b) {
  return a + b;
}
Aquí, la firma de la función es `sumar(a, b)`.

¿Qué es un retorno de una función en JavaScript?

El retorno de una función es el valor que la función devuelve cuando termina su ejecución. En JavaScript se usa la palabra reservada `return` para enviar ese valor al lugar donde se llamó la función. Ejemplo:
function obtenerEdad() {
  return 25;
}
Aquí, el retorno de la función es `25`.

¿Qué es un argumento en JavaScript?

Un argumento es el valor que se pasa a una función cuando se la llama. Esos valores se asignan a los parámetros de la función. Ejemplo:
function saludar(nombre) {
  return `Hola, ${nombre}`;
}
saludar('Ana');
En este caso, `'Ana'` es el argumento.
